Bloquea las cookies hasta que el usuario las acepte para cumplir la normativa acerca de las cookies.

== Description ==

Este plugin bloquea todas las coockies as&#237; que si tenemos una tienda online, por ejemplo y el usuario no lo acepta es posible que encontremos errores en el funcionamiento de cualquier proceso que requiera del uso de estas. 

El plugin es gratuito y todos los comentarios para mejorarlo ser&#225; bienvenidos.

Pixelpro no se hace responsable por da&ntilde;os, perjuicios, reclamaciones o gastos derivados del uso de este plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. sube el archivo `cookiespro.zip` al directorio de tu Wordpress `/wp-content/plugins/`
1. Activa el plugin a través del menú 'Plugins' de tu WordPress

== Frequently Asked Questions ==

= ¿Bloquea todas las cookies? =

Si a excepción de las que utiliza Wordpress para iniciar sesión y que son importantes, incluidas las de Google Analitycs, hasta que el usuario le de a aceptar.

= ¿Puedo personalizar el banner? =

Completamente y añadir tus propios estilos a través del css que se encuentra en el plugin.
